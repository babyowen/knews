# KNews - 关键词新闻AI摘要系统

## 项目简介
KNews 是一个基于关键词的新闻AI摘要系统。它可以根据用户选择的关键词和日期，自动获取相关新闻并生成AI摘要。系统支持关键词分类管理，让用户可以更方便地选择和管理关键词。

## 主要功能
1. 关键词分类管理
2. 日期选择
3. 新闻AI摘要
4. 语音播报摘要内容
